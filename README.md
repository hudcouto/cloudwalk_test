# CloudWalk test

## dependencies

- ruby 3.1.1

## how to install

1. Install bundler

``` sh
gem install bundler
```

2. install dependencies

``` sh
bundle install
```

## how to run

``` sh
ruby app.rb
```

> Obs: after that a file called report.json wil be generated

## run specs

``` sh
rspec
```
