# CloudWalk test

## dependencies

- ruby 3.1.1

## how to install

1. Install bundler

``` sh
gem install bundler
```

2. install dependencies

``` sh
bundle install
```

## how to run

``` sh
ruby app.rb
```

> Obs: after that two files called default_report.json and kills_by_death_type_report.json wil be generated

## run specs

``` sh
rspec
```

## running using docker

``` sh
docker-compose up
```
